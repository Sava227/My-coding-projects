Попробуйте наш кофе

Американо

Классика жанра. Не слишком крепкий, зато объемный и долгий в удовольствии. Идеален с молоком или сливками.

Банановый капучино

Необычное и очень вкусное сочетание банана и кофе. Мягкий вкус, легкий аромат - для тех, кто любит эксперименты.

Фраппучино

Холодный кофейный десерт со льдом, молоком и взбитыми сливками. Отличный выбор в жару.

Латте

Нежный кофейно-молочный напиток с бархатистой пенкой. Идеально для неспешного утра.

Мокко

Кофе + шоколад + молоко. Для тех, кто не может выбрать между кофе и горячим шоколадом.
