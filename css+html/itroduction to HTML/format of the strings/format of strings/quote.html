Одна из самых популярных цитат про программирование:

Есть два способа создания дизайна программы. Один из них, это сделать его настолько простым, что, очевидно, не будет недостатков. Другой способ — сделать его настолько запутанным, что в нем не будет очевидных недостатков.

Также среди программистов известна цитата: Не волнуйся, если не работает. Если бы всё всегда работало, у тебя бы не было работы.
